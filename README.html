Context

This project takes existing data from The COVID Racial Data Tracker and reforms it to display a comparison between Oregon and other states and territories.

The genesis of this project was this email message:

On Jul 1, 2020, at 9:50 AM, Allen Patrick Patrick.Allen@dhsoha.state.or.us wrote:

Jeremy,

Do you have anyone who could spend a little time on a data analysis project for me? The link below is to racial disparity data from covid. I need someone to a) validate that we agree with the reported data (some of the data looks a little off to me), and then b) figure out a way to display or describe the data in a way that benchmarks our disparities against other states.

Here's the link:

https://covidtracking.com/race/dashboard

Let me know if you’ve got someone to hand this off to. Thanks!

Pat.

Project status

Active development on pause since late-July 2020. Data updates and dashboard publication are currently maintained (Mondays and Thursdays).

Build

  1. Run the make.bat script from the Windows command line
    1. The script executes mungeData.R, which creates the source data file, Data\disparity_data.csv, for the Tableau workbook
    2. The data file is timestamped and version controlled
  2. Publish Covid19 Racial and Ethnic Disparity.twb to the internal Tableau server
